 Connectopia — Lead Manager CRM

A full-stack CRM web application to manage client leads efficiently.

🚀 Features
- Admin login authentication
- Add, edit, delete leads
- Status tracking (New, Contacted, Closed)
- Search and filter leads
- Dashboard analytics
- MongoDB database integration

🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

🔧 Setup
1. Clone the repo
2. Install dependencies:
   npm install
3. Start server:
   node server.js
4. Open frontend/login.html
